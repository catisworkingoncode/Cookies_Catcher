# Cookie Catcher

A colorful browser game for kids where treats fall from the sky and the player moves a basket to catch them.

## How To Play

Open `index.html` in a browser.

- Use the left and right arrow keys to move the basket.
- Catch positive items to gain points.
- Avoid negative items, or catch them and lose points plus one life.
- The game ends when lives reach 0.
- Use the pause menu to resume, restart, mute, or adjust volume.

## Scoring

- 🧋 Bubble tea: +5
- 🍔 Burger: +3
- 🍪 Cookie: +1
- 🔥 Fire: -1
- 💣 Bomb: -3
- 🦠 Germ: -5

## Features

- Vanilla HTML, CSS, and JavaScript
- Runs directly by opening `index.html`
- Kid-friendly interface
- Score, level, lives, and best score
- Increasing difficulty as score rises
- Pause menu
- Background music
- Positive and negative catch sound effects
- Adjustable volume
- Best score saved with `localStorage`
